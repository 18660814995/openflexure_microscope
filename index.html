Openflexure microscope

An open-source 3D printable microscope, complete with high-precision XYZ stage.

Download this project as a .zip file

3D Printed Microscopes!

This project has been going for a couple of years now, aiming to make a microscope that not only has high-magnification optics but also a really good mechanical stage. Development is ongoing, but it's quite well optimised already, with a study on its mechanical performance and a stable release. This site is a little sparse at present, but the repository now has all the source files for the microscope.

Projects based on OpenFlexure Microscope

This design has been used in lots of different places, from labs to schools - but one initiative I'm very keen to support is WaterScope, a not for profit start-up aiming to make testing for bacterial contamination much faster, cheaper, and easier to use.
